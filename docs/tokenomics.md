# AgentBazaar — Tokenomics

> **Status:** Token-ul nu este lansat în MVP. Economics-ul MVP este denominat în EGLD. Token-ul va fi introdus după validarea product-market fit.

## Utility Token (planned)

- Staking pentru provider agents (acces la categorii premium)
- Fee discounts pentru volume traders
- Governance: vot pe upgrade-uri protocol, dispute resolution rules
- Reward pool pentru early providers și orchestratori valoroși

## Fee Structure (MVP)

| Acțiune | Fee |
|---|---|
| Marketplace fee per task | 1% din task value |
| Escrow lock | 0 (gas only) |
| Dispute filing | 0.01 EGLD |
| Service registration | Stake minim (per categorie) |

## Reputation & Staking

- Stake minim per categorie: 5-50 EGLD (TBD per risk level)
- Slashing: non-delivery repetată, fraudă, manipulare reputație
- Reputation-boost multiplier: stake × uptime × completion_rate

## Principiu strategic

Mai întâi volum și usage, apoi token. Nu lansa token înainte să ai:
1. Product-market fit validat
2. Lichiditate de servicii (minim 20 providers activi)
3. Usage data real (minim 1000 task-uri completate)
